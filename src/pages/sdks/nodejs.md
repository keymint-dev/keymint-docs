# KeyMint Node.js SDK

Documentation for the official KeyMint Node.js SDK.

Install the SDK from npm:

```bash
npm install keymint-nodejs
```

Stay tuned for more detailed documentation, including usage examples and API reference.

You can find the SDK on [npm](https://www.npmjs.com/package/keymint-nodejs) and on [GitHub](https://github.com/keymint-dev/KeymintSDK-nodejs).
