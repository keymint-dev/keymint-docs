<Info>
The official Keymint Python SDK provides a modern, intuitive interface for integrating license management into your Python applications with full type hint support.
</Info>

## Features

<CardGroup cols={2}>
  <Card title="Type Hints" icon="code">
    Full type hint support for better IDE integration and code safety
  </Card>
  <Card title="Comprehensive Coverage" icon="shield-check">
    Complete API coverage for all Keymint endpoints
  </Card>
  <Card title="Pythonic Interface" icon="python">
    Clean, intuitive API that follows Python conventions
  </Card>
</CardGroup>

## Installation

<Tabs>
  <Tab title="pip">
    ```bash
    pip install keymint
    ```
  </Tab>
  
  <Tab title="Poetry">
    ```bash
    poetry add keymint
    ```
  </Tab>
  
  <Tab title="pipenv">
    ```bash
    pipenv install keymint
    ```
  </Tab>
  
  <Tab title="requirements.txt">
    ```text
    keymint>=1.0.0
    ```
  </Tab>
</Tabs>
